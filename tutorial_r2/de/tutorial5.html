Einstellungen

Ein kurzer Klick auf den Schaltknopf Einstellungen öffnet den Einstellungs-Bildschirm. Hier können sie einige Eigenschaften von Meine Ausgaben konfigurieren, und haben auch Zugriff auf die Kontoverwaltung und die Backup-Funktion. Ein langer Klick öffnet ein Menü mit schnellem Zugriff auf die Konfiguration des aktuellen Kontos, sowie aufs Backup.

Verwaltung der Konten

Der Eintrag Konten öffnet den Bildschirm , der eine komplette Liste der Konten anzeigt. Durch den Klick auf ein Konto wird das Konfigurationsformular geöffnet (erklärt in Konten ). Ein langer Klick bringt das Kontextmenü zum Vorschein, mit der Möglichkeit, das Konto zu löschen, mit der Ausnahme des aktuellen Kontos, das nicht gelöscht werden kann.

Sortierreihenfolge der Kategorien

Standardmäßig werden die Kategorien anhand der Benützungshäufigkeit sortiert (wie oft werden sie einer Buchung zugeordnet). Entfernen Sie das Häckchen von dieser Option, wenn Sie möchten, dass die Kategorien immer in alphabetischer Reihenfolge aufgelistet werden.

Upload URL QIF Export

Wenn Sie die URL eines Ordners angeben, auf den Sie über FTP Schreibzugriff haben (z.B. "ftp://login:password@my.example.org:port/my/directory/"), wird Meine Ausgaben die QIF Datei stets nach dem Export der Buchungen dorthin übertragen. Wenn Sie eine Email URL (z.B. "mailto:john@my.example.com") angeben, wird Ihre Email-Anwendung gestartet, und eine Email erzeugt mit der Datei als Anhang.

Backup und Wiederherstellung

Backup erzeugt eine Sicherungskopie von Datenbank und Einstellungen im myexpenses Ordner auf der SD-Karte. Die Datenbank wird in der Datei BACKUP gespeichert, die Einstellungen in der Datei BACKUP_PREF. Beide werden automatisch wiederhergestellt, wenn Meine Ausgaben neu installiert wird. Sie können also die Anwendungsdaten komplett auf ein weiteres Gerät kopieren, indem Sie diese beiden Dateien an denselben Ort auf diesem kopieren, bevor Sie Meine Ausgaben installieren.

Import aus Grisbi

Dieser Eintrag erlaubt die in Grisbi definierten Kategorien (Englisch, Deutsch oder Französisch) zu importieren. Sie können auch eine Grisbi Datei (in XML) verwenden, die Ihre eigenen Kategorien enthält. Sie können diese Datei editieren und alle Informationen außer die Elemente "Category" und "Sub_category" entfernen[1]. Kopieren Sie diese Datei unter dem Namen categories.xml in einen Ordner myexpenses in das Wurzelverzeichnis der SD-Karte.


[1] Meine Ausgaben unterstützt auch das Grisbi Datei Format 0.5.0, in dem die Elemente Categorie und Sous_categorie heißen.
