Domande più frequenti

1. Posso inviare il file QIF di Google Docs o Dropbox?
2. Quando inserisco una nuova operazione, non trovo il campo per selezionare il metodo di pagamento. Perchè?
3. Come posso ripristinare il backup?
4. Chiave contrib, che cos'è?
5. Come faccio a importare le categorie in gruppo?

1.

Posso inviare il file QIF di Google Docs o Dropbox?

Sì, nelle parametri, attivare Condividi Export, e lasciare Carica URL Export in bianco. Così, dopo ogni esportazione, è possibile scegliere da un menu un applicazioni (ad esempio Google Docs o Dropbox) in grado di trasferire il file.

2.

Quando inserisco una nuova operazione, non trovo il campo per selezionare il metodo di pagamento. Perchè?

Il metodo di pagamento è legato al tipo di conto. Nell'impostazione predefinita, non ci sono metodi di pagamento per il conto contanti, solo per i conti bancari. E' possibile cambiare un tipo di conto in Modificare il conto, oppure creare un nuovo metodo di pagamento (Impostazioni → Gestione → Modi di pagamento → Nuovo modo di pagamento).

3.

Come posso ripristinare il backup?

Se crea un backup, questo viene memorizzato nella cartella myexpenses della vostra scheda SD. Esso verrá scoperto quando si disinstalla e reinstalla Le mie spese, e si puó scegliere di ripristinarlo. Dalla versione 1.7.0. è possibile attivare il ripristino dalla schermata Impostazioni (Funzionalità speciale della versione Contrib).

4.

Chiave contrib, che cos'è?

Le mie spese è distribuito con alcune funzionalità premium che possono essere provate cinque volte e quindi possono essere sbloccate con l'installazione della app contrib a pagamento da Google Play. Potrebbe essere necessario lanciare Le mie spese una volta dalla schermata iniziale del contrib applicazione, al fine di renderlo consapevole delle app contrib. Ma c'è bisogno di alcuna chiave contrib in quel caso. Una chiave contrib è un meccanismo per sbloccare le funzionalità Premium senza installazione delle app contrib. Manderò questa chiave agli utenti che contribuiscono allo sviluppo delLe Mie Spese con altri mezzi, ad esempio traducendo l'interfaccia utente, testando pre-releases o una donazione tramite Paypal o Flattr. La chiave è calcolata sulla base della Android-ID del dispositivo, che potete comunicare a me attraverso la voce Richiede una chiave contrib nella schermata Impostazioni.

5.

Come faccio a importare le categorie in gruppo?

Le mie spese può importare le categorie dal file Grisbi. Conservare questo file come grisbi.xml in una cartella myexpenses nella cartella principale della vostra scheda SD e chiamare la voce Importazione dei dati Grisbi nella schermata Impostazioni. Hai anche la possibilità di importare i beneficiari Grisbi registrati nello stesso file (elemento Party). Se si sceglie il proprio file come fonte di importazione, è possibile importare categorie e beneficiari, o solo categorie. Il beneficiario sarà offerta automaticamente per riempire il "Beneficiario" in forma di operazioni. Se si crea il proprio file XML, rispettare la seguente struttura: 


<root>
  <Category Nb="1" Na="Categoria principale A" />
  <Sub_category Nbc="1" Nb="1" Na="Sottocategoria A.1" />
  <Sub_category Nbc="1" Nb="2" Na="Sottocategoria A.2" />
  <Category Nb="2" Na="Categoria principale B" />
  <Sub_category Nbc="2" Nb="1" Na="Sottocategoria B.1" />
  <Sub_category Nbc="2" Nb="2" Na="Sottocategoria B.2" />
  <Party Na="N.N." />
</root>
