Häufig gestellte Fragen

1. Kann ich die QIF Datei auf Google Docs oder Dropbox übertragen?
2. Wenn ich eine neue Buchung anlege, fehlt das Feld für die Zahlungsweise. Warum?
3. Wie kann ich ein Backup wiederherstellen?
4. Wie kann ich Kategorien im Batch importieren?

1.

Kann ich die QIF Datei auf Google Docs oder Dropbox übertragen?

Ja, aktivieren Sie Export versenden in den Einstellungen, und lassen Sie Upload URL leer. Dann können Sie nach jedem Export die App (wie z.B. Google Docs oder Dropbox) auswählen, mittels deren Sie die Datei übertragen wollen.

2.

Wenn ich eine neue Buchung anlege, fehlt das Feld für die Zahlungsweise. Warum?

Zahlungsweisen sind den Kontotypen zugeordnet. Zu Beginn sind nur für Bankkonten Zahlungsweisen definiert, nicht für Bargeldkonten. Ändern Sie den Kontotyp unter Konto verändern, oder definieren Sie eine neue Zahlungsweise (Einstellungen → Verwalten → Zahlungsweisen → Neue Zahlungsweise).

3.

Wie kann ich ein Backup wiederherstellen?

Wenn Sie ein Backup erzeugen, wird dieses im Ordner myexpenses auf der SD-Karte gespeichert. Dort wird es entdeckt, wenn Sie Meine Ausgaben deinstallieren und erneut installieren, und Sie können die Wiederherstellung wählen. Seit Version 1.7.0, kann die Wiederherstellung auch vom Einstellungsbildschirm aus aufgerufen werden.

4.

Wie kann ich Kategorien im Batch importieren?

Meine Ausgaben kann Kategorien aus Ihrer Grisbi Datei importieren. Kopieren Sie diese Datei unter dem Namen grisbi.xml in einen Ordner myexpenses in das Wurzelverzeichnis der SD-Karte und rufen Sie den Eintrag Import aus Grisbi in den Einstellungen auf. Sie können zusätzlich auch die von Grisbi gespeicherten Parteien (Empfänger und Zahler), aus dieser Datei importieren. Wenn Sie Ihre eigene Datei als Import-Quelle auswählen, können Sie entweder Parteien und Kategorien oder nur Kategorien importieren. Die Partner werden im Buchungsformular verwendet, um Ihnen für das Feld Empfänger/Zahler ein automatische Ausfüllen vorzuschlagen. Wenn Sie Ihre eigene XML Datei erstellen, halten Sie sich an die folgende Struktur: 


<root>
  <Category Nb="1" Na="Hauptkategorie A" />
  <Sub_category Nbc="1" Nb="1" Na="Unterkategorie A.1" />
  <Sub_category Nbc="1" Nb="2" Na="Unterkategorie A.2" />
  <Category Nb="2" Na="Hauptkategorie B" />
  <Sub_category Nbc="2" Nb="1" Na="Unterkategorie B.1" />
  <Sub_category Nbc="2" Nb="2" Na="Unterkategorie B.2" />
  <Party Na="N.N." />
</root>
