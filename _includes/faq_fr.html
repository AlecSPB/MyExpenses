Foire aux questions

1. Puis-je envoyer le fichier QIF vers Google Docs ou Dropbox?
2. Lors de la création d'une nouvelle transaction, il n'existe pas de champ pour le mode de paiement. Pourquoi?
3. Comment puis-je récupérer ma sauvegarde ?
4. Comment puis-je importer les catégories en batch?

1.

Puis-je envoyer le fichier QIF vers Google Docs ou Dropbox?

Oui, dans les paramètres, activez Envoyer le fichier d'export, et gardez URL où sera téléchargé le fichier d'export vide. Ainsi, après chaque export, vous pourrez choisir dans un menu une apps (par exemple Google Docs ou Dropbox) capable de transférer le fichier.

2.

Lors de la création d'une nouvelle transaction, il n'existe pas de champ pour le mode de paiement. Pourquoi?

Les modes de paiement sont définis par types de comptes. Par défaut, il n'existe pas de méthodes de paiement pour les comptes de caisse, uniquement pour les comptes bancaires. Vous pouvez modifier le type du compte par Éditer le compte, ou définir un nouveau mode de paiement (Paramètres → Gestion → Méthodes de paiement → Nouvelle méthode de paiement).

3.

Comment puis-je récupérer ma sauvegarde ?

Lorsque vous créez une sauvegarde, celle-ci est stockée dans le répertoire myexpenses de votre carte SD. Elle est détectée lorsque vous désinstallez puis réinstallez Mes dépenses, et vous pouvez choissir de la recupérer. À partir de la version 1.7.0., la recupération de la sauvegarde peut aussi être initiée à partir des paramètres.

4.

Comment puis-je importer les catégories en batch?

Mes dépenses peut importer les catégories à partir de votre fichier Grisbi. Copiez celui-ci sous le nom grisbi.xml dans le répertoire myexpenses de la racine de votre carte SD et appelez l'entree Import des données Grisbi dans l'écran Paramètres. Vous avez aussi la possibilité d'importer les tiers tel que Grisbi les enregistre dans ce même fichier. Si vous choississez votre propre fichier en tant que source de l'import, vous pourrez soit importer tiers et catégories, soit uniquement les catégories. Les tiers vous seront proposés pour remplir automatiquement le champ "tiers" dans le formulaire des opérations. Si vous éditez votre propre fichier XML, respectez la structure suivante: 


<root>
  <Category Nb="1" Na="Catégorie principale A" />
  <Sub_category Nbc="1" Nb="1" Na="Sous-catégorie A.1" />
  <Sub_category Nbc="1" Nb="2" Na="Sous-catégorie A.2" />
  <Category Nb="2" Na="Catégorie principale B" />
  <Sub_category Nbc="2" Nb="1" Na="Sous-catégorie B.1" />
  <Sub_category Nbc="2" Nb="2" Na="Sous-catégorie B.2" />
  <Party Na="N.N." />
</root>
