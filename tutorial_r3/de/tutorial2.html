Buchungen

Kurzer Klick auf Neu öffnet den Bildschirm für die Eingabe einer neuen Buchung.

Passen Sie Datum und Uhrzeit an, klicken Sie auf den Schaltknopf - um zwischen Ausgabe und Einnahme zu wechseln, geben Sie Betrag[1] und Notizen ein. Das Feld für die Eingabe von Empfänger (bzw. Zahler) merkt sich die von Ihnen eingegebenen Werte, und schlägt diese bei weiteren Eingaben in diesem Feld vor[2]. Durch Klick auf den Knopf Auswählen kommen Sie zum Bildschirm Kategorie auswählen. Dieser wird später erklärt.

Falls dem Typ des aktuellen Kontos Zahlungsweisen zugeordnet sind, können Sie durch den Schaltknopf Auswählen neben Zahlungsweise die Zahlungsweise der Buchung definieren.

In der Liste der Buchungen, kann jede Buchung durch einen kurzen Klick bearbeitet werden, das Kontextmenü erlaubt das Löschen und die Anzeige von Detailinformationen (Empfänger, Notizen). Sie können eine Buchung auch in ein anderes Konto verschieben, sofern sie zumindest ein weiteres Konto angelegt haben.

Umbuchung

Neu → Umbuchung erlaubt Beträge zwichen Konten zu verschieben. Dieser Befehl steht nur zur Verfügung, wenn es mehrere Konten mit derselben Währung gibt.


[1] Das Eingabeformat für Währungsbeträge kann mittels Dezimaltrennzeichen für die Eingabe von Währungsbeträgen eingestellt werden.

[2] Sie können auch die in einer Grisbi Datei gespeicherte Liste der Parteien importieren (erklärt in Import aus Grisbi). Wie Sie Werte aus dieser Liste löschen können, erfahren Sie unter Parteien verwalten.
