Parametri

Un breve clic sul pulsante Parametri si apre la schermata parametri. Qui è possibile impostare alcune proprietà di applicazione Le mie spese può accedere alla gestione degli account e di backup. Un clic lungo si apre un menu che include l'accesso diretto al conto attuale (Conto), e di backup (Backup).

Gestione dei conti

Questo mostra l'elenco completo dei conti. Facendo clic sul conto, accedere al modulo di installazione (spiegato nel Conti ). Un clic lungo visualizza un menu di scelta rapida per cancellare conti, ad eccezione del conto corrente, che non possono essere cancellati.

Criterio di ordinamento per le categorie

Per impostazione predefinita, le categorie sono ordinati per frequenza di utilizzo (numero di volte che sono normalmente associati con una transazione). Deselezionare questa opzione se si preferisce che le categorie sono visualizzate in ordine alfabetico.

Condividi QIF Export

Se si seleziona questa opzione, il file QIF verrà caricato dopo ogni esportare i vostri transazioni. Sotto Carica URL QIF Export, è necessario specificare la destinazione:

  • L'URL di un file tramite FTP in modalità di scrittura (ad esempio "ftp://login:password@my.example.org:port/my/directory/").

  • un URL per email (ad esempio "mailto:john@my.example.com"). Il vostro applicazione di posta elettronica si aprirà con un nuovo messaggio con il file allegato.

  • Se questo campo è vuoto, è possibile scegliere ogni volta che le applicazioni disponibili sul dispositivo, in grado di trasferire un file (ad esempio, Dropbox, Google docs, …).

Backup e ripristino

Backup crea un backup del database e delle preferenze nella cartella myexpenses sulla scheda SD. Il database viene memorizzato nel file chiamato BACKUP e le preferenze nel file BACKUP_PREF. Entrambi sono automaticamente ripristinate quando si reinstalla Le mie spese. In questo modo è possibile recuperare tutte le vostre applicazioni dati su un dispositivo diverso da copiare questi due file nella stessa posizione là, prima di installare Le mie spese.

Importazione dei dati Grisbi

Importare le categorie permette di impostare le categorie di file inclusa da Grisbi in inglese, tedesco o francese. Se con Grisbi hai già definite le proprie categorie, è possibile anche fornire il file Grisbi, che è in XML. Se si desidera, è possibile modificare il file con un editor e rimuovere ogni altra informazione rispetto alla categoria (Category) e gli elementi Sub_category[3]. Conservare questo file come grisbi.xml in un myexpenses cartella nella directory principale della vostra scheda SD[4]. Hai anche la possibilità di importare i beneficiari Grisbi registrati nello stesso file (elemento Party). Se si sceglie il proprio file come fonte di importazione, è possibile importare categorie e beneficiari, o solo categorie. Il beneficiario sarà offerta automaticamente per riempire il "Beneficiario" in forma di operazioni.

Utilizzare Android standard menu

Se si seleziona questa opzione, invece di visualizzare una barra dei pulsanti, sarà necessario utilizzare il menu standard per accedere al Android ordini. È inoltre possibile accedere alla modifica della configurazione semplicemente attivando il pulsante Menu hardware.

Gestione dei beneficiari

Su questo schermo l'elenco dei terzi utilizzato per il completamento automatico del campo Pagatore/Beneficiario. Per rimuovere le voci da questo elenco, selezionarle controllando il loro diritto, e quindi attivare il pulsante Cancellare.

Separatore decimale per l'ingresso della quantità

Per impostazione predefinita, gli importi delle transazioni (o il saldo di apertura di un conto) utilizzare il separatore decimale delle impostazioni locali (virgola in molti paesi europei, il punto in più paesi di lingua inglese)[5], qui si può configurare in modo esplicito. In alternativa si ha la possibilità di ingresso alle unità minori (es. $ centesimi, € cent o penny £) come interi (Unità minori come numeri interi ).

Gestione dei modi di pagamento

Su questo schermo si vede la lista dei modi di pagamento predefiniti (assegno, carta di credito, pagamento, addebito) e qui puó definire nuovi modi di pagamento. Si fa clic su un modo della lista (o sceglie Nuovo), arriva a un formulario, dove può definire la descrizione (salvo per i predefiniti) e il tipo (si tratta di un modo di pagamento per entrate o spese o neutrale), e scegliere i tipi di conto per i quali è applicabile il modo di pagamento.
