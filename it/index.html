Le mie spese è stato progettato per tenere traccia delle spese, e per esportare queste spese come file QIF in un programma di finanziamento (es. Grisbi (Open Source), Gnucash (Open Source), MS Money, Quicken o Wiso Mein Geld.
Richiede Android 2.0 e superiori.

Caratteristiche:

  • piú conti con trasferimenti
  • due livelli di categorie (importazione da Grisbi XML)
  • caricare file QIF via FTP, e-mail o Dropbox, ...
  • backup e ripristino di dati
