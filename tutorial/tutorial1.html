Managing accounts

Gestion des comptes

Verwaltung der Konten

The first thing you will probably want to do, is edit the default account, or create a new one. Select the "Select Account" entry from the menu.
En premier lieu, vous voudrez certainement modifier le compte par défaut ou en créer un nouveau. Activez l'entrée "Sélectionner un compte" du menu.
Als erstes möchten Sie wahrscheinlich das bei der Installation eingerichtete Konto verändern, oder ein neues Konto einrichten. Aktivieren Sie den Eintrag "Konto auswählen" im Menü.
As in other Android applications, a long click on the account brings up the context menu, from where you can edit or delete the account. The current account cannot be deleted.
Comme toute application Android, un clic long sur le compte vous ouvre un menu contextuel, avec lequel vous pourrez éditer ou supprimer celui-ci. Le compte actuellement sélectionné ne peut pas être supprimé.
Wie in anderen Android Apps, wird durch einen langen Klick auf ein Konto ein Kontextmenü aufgerufen mit Befehlen zum Verändern oder Löschen. Das aktuell ausgewählte Konto kann nicht gelöscht werden.
Enter a label, a description, the opening balance, and a currency for the account.
Entrez un libellé, une description, le solde d'ouverture et la devise de ce compte.
Geben Sie eine Bezeichnung, eine Beschreibung, den Anfangssaldo und die Währung des Kontos ein.
Confirm that you are done, and you get back to the list of accounts. You can set up as many accounts as you like through the menu entry "Create new account". Selecting the current account is done through a short click on it in the list.
Confirmez et vous reviendrez à la liste des comptes. Vous pourrez entrer autant de comptes que vous le désirez avec le menu "Créer un nouveau compte". Pour sélectionner un compte, il suffit d'un clic court sur celui-ci dans la liste.
Bestätigen Sie die Eingabe, und Sie gelangen zurück zur Kontoliste. Der Menüeintrag "Neues Konto erstellen" erlaubt beliebig viele Konten anzulegen. Durch einen kurzen Klick auf ein Konto in der Liste, wird dieses als aktuelles Konto ausgewählt.
