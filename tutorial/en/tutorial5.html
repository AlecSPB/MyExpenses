Backup and restore

Backup creates a backup of database and preferences in the myexpenses folder on the sdcard. The database is stored in a file named BACKUP, the preferences in a file named BACKUP_PREF. Both are automatically restored when you reinstall MyExpenses. Thus you can retrieve all your application data on a different device by copying these two files into the same location there, before installing MyExpenses.
