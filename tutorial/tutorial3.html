Managing categories

Gestion des catégories

Verwaltung der Kategorien

At the beginning, no categories are defined. You have two options for adding them, both accessible from the menu.
Au départ, aucune catégorie n'est prédéfinie. Vous avez deux possibilités à partir du menu pour les ajouter.
Zu Beginn gibt es noch keine Kategorien. Vom Menü aus gibt es zwei Möglichkeiten, diese anzulegen.
"Import categories" allows to set up categories from files included from Grisbi in English, German or French. If with Grisbi you have already defined your own categories, you can also provide your Grisbi file, which is in XML. If you want, you can edit the file with an editor and remove any information other than the Category and Sub_category elements [1]. Store this file as categories.xml in a folder "myexpenses" in the root of your sdcard.
"Importer les catégories" vous permet de les entrer directement à partir des définitions standards de Grisbi en anglais, allemand ou français. Si vous aviez déjà défini vos propres catégories, vous pouvez aussi utiliser votre fichier Grisbi qui est en format XML. Dans ce cas, copiez votre fichier et enlevez toutes les informations autres que les blocs de catégories (Category) et de sous-catégories (Sub_category) [1]. Sauvegardez ce fichier sous le nom "categories.xml" dans le répertoire "myexpenses" de la racine de votre carte SD.
"Kategorien importieren" erlaubt die in Grisbi definierten Kategorien (Englisch, Deutsch oder Französisch) zu importieren. Sie können auch eine Grisbi Datei (in XML) verwenden, die Ihre eigenen Kategorien enthält. Sie können diese Datei editieren und alle Informationen außer die Elemente "Category" und "Sub_category" entfernen [1]. Kopieren Sie diese Datei unter dem Namen categories.xml in einen Ordner "myexpenses" in das Wurzelverzeichnis der SD-Karte.
You now have a set up categories with subcategories you can associate with your transactions.
Vous avez maintenant un échantillon de catégories et de sous-catégories que vous pouvez utiliser pour vos opérations.
Sie können die importierten Kategorien und Unterkategorien jetzt Ihren Buchungen zuordnen.
The second option is to add your own categories.  "Create new main category" brings up a dialog for providing a new category label.
La seconde façon est d'ajouter vos propres catégories. "Nouvelle catégorie principale" vous ouvre une fenêtre pour entrer un nouveau libellé.
Die zweite Möglichkeit besteht darin, Kategorien selbst anzulegen. "Neue Hauptkategorie" öffnet einen Dialog zur Eingabe einer Bezeichnung.
Clicking on a main category, opens up its subcategories. You select a subcategory for mapping to the current transaction by simply clicking on it. In order to map a main category, you have to call its context menu.
Cliquez sur une catégorie principale pour ouvrir ses sous-catégories. Vous pouvez sélectionner une sous-catégorie pour l'opération en cours en cliquant simplement dessus. Si vous voulez utiliser la catégorie principale, il vous faudra passer par le menu contextuel.
Ein Klick auf eine Hauptkategorie öffnet die Liste der Unterkategorien. Die Zuordnung einer Unterkategorie zur aktuellen Buchung erfolgt durch kurzen Klick. Die Zuordnung einer Hauptkategorie ist durch das Kontextmenü möglich.
From here you can also add new subcategories, delete a category or edit it. Subcategories have also context menus for deleting and editing.
A partir de là, vous pouvez ajouter de nouvelles sous-catégories, effacer ou modifier une catégorie principale. Les sous-catégories ont aussi un menu contextuel afin de les effacer ou de les modifier.
Dieses erlaubt auch das Anlegen neuer Unterkategorien, das Löschen und das Verändern einer Hauptkategorie. Die Unterkategorien verfügen auch über ein Kontextmenü mit Einträgen zum Löschen und Verändern.
[1] MyExpenses also support the Grisbi file format 0.5.0, where the elements are named Categorie and Sous_categorie. ↩
[1] MyExpenses supporte aussi la version 0.5.0 du format de fichier Grisib, où les éléments portent les noms Categorie et Sous_categorie (sic!). ↩
[1] MyExpenses unterstützt auch das Grisbi Datei Format 0.5.0, in dem die Elemente Categorie und Sous_categorie heißen. ↩
