Gestione delle categorie

All'inizio, categorie non sono definite. Sono disponibili due opzioni per l'aggiunta di loro, entrambi accessibili dal menù.

Importare le categorie permette di impostare le categorie di file inclusa da Grisbi in inglese, tedesco o francese. Se con Grisbi hai già definite le proprie categorie, è possibile anche fornire il file Grisbi, che è in XML. Se si desidera, è possibile modificare il file con un editor e rimuovere ogni altra informazione rispetto alla categoria e gli elementi Sub_category.[1]Conservare questo file come categories.xml in un myexpenses cartella nella directory principale della vostra scheda SD.

Ora avete un elenco di categorie con sottocategorie che è possibile associare con il tuo transazioni.

La seconda opzione è quella di aggiungere proprie categorie. Nuova categoria principale porta in primo piano una finestra di dialogo per la fornitura di una nuova etichetta di categoria.

Cliccando su una categoria principale, apre le sue sottocategorie. Si seleziona una sottocategoria per la transazione corrente, semplicemente cliccando. Al fine di mappare un categoria principale, si deve chiamare il suo menù contestuale.

Da qui è inoltre possibile aggiungere nuove sottocategorie, eliminare una categoria o modificarlo. Sottocategorie hanno anche menù di scelta rapida per la eliminazione e la modifica.


[1] Le mie spese supporta anche il formato di file Grisbi 0.5.0, dove gli elementi sono denominati Categorie e Sous_categorie.
