Esportazione di operazioni

Infine, la ragion d'essere di Le mie spese è il suo supporto per l'esportazione in formato QIF. Chiama questa funzione attraverso la voce di menù dalla lista transazione. Questo sarà crea un file sul tuo scheda SD con tutte le transazioni per il conto corrente, eliminare dal database, e regolare il saldo di apertura con il saldo corrente. Le mie spese posso condividere il file subito dopo il export, sia caricarlo in un server FTP o inviarla via e-mail. È possibile configurare il URL dalla voce di menù "Parametri" (ad esempio "ftp://login:password@my.example.org:port/my/directory/" o "mailto:john@my.example.com").
Operazioni di trasferimento verrà eliminato anche dal conto, ma ancora visibile in l'altro conto a cui si riferiscono.

Voilà. È ora possibile collegare il telefono al computer, aprire Grisbi (o GnuCash), e utilizzare la funzionalità di importazione.
