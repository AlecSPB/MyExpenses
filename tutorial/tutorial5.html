Backup und Wiederherstellung

"Backup" creates a backup of database and preferences in the "myexpenses" folder on the sdcard. The database is stored in a file named "BACKUP", the preferences in a file named "BACKUP_PREF". Both are automatically restored when you reinstall "My Expenses". Thus you can retrieve all your application data on a different device by copying these two files into the same location there, before installing "My Expenses".
"Sauvegarde" créé une sauvegarde de la base de données et des préférences dans le répertoire "myexpenses" de la carte SD. La base de données est stockée dans le fichier "BACKUP" et les préférences dans "BACKUP_PREF". Ceux-ci sont automatiquement restaurés lorsque vous réinstallez "My Expenses". Ainsi, vous pouvez récupérer toutes vos données sur un périphérique différent en copiant ces deux fichiers au même endroit avant d'installer "My Expenses".
"Backup" erzeugt eine Sicherungskopie von Datenbank und Einstellungen im "myexpenses" Ordner auf der SD-Karte. Die Datenbank wird in der Datei "BACKUP" gespeichert, die Einstellungen in der Datei "BACKUP_PREF". Beide werden automatisch wiederhergestellt, wenn "My expenses" neu installiert wird. Sie können also die Anwendungsdaten komplett auf ein weiteres Gerät kopieren, indem Sie diese beiden Dateien an denselben Ort auf diesem kopieren, bevor Sie "My Expenses" installieren.
