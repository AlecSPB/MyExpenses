Resum

Les Meves Despeses està dissenyat per controlar les despeses, i exportar-les com fitxers QIF a un programa de finances (per ex.: Grisbi, GnuCash, MS Money, Quicken).

Característiques

  • Múltiples comptes amb transferències.
  • Genera transaccions a partir de plantilles
  • Dos nivells de categories (es poden importar de Grisbi XML)
  • Enviament del fitxer QIF via FTP o correu
  • Còpia de seguretat i restauració de les dades
