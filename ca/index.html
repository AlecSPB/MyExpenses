Resum

  • Mantingui un registre de les seves despeses i ingressos.
  • Administrar directament en el seu telèfon intel · ligent.
  • Exportar-a un programa de finançament d'escriptori com Grisbi (Open Source), Gnucash (Open Source), Quicken o Microsoft Money.
  • Funciona en totes les versions d'Android 2.0 a 4.2.

Característiques

  • Múltiples comptes amb transferències.
  • Genera transaccions a partir de plantilles
  • Dos nivells de categories (es poden importar de Grisbi XML)
  • Enviament del fitxer export via FTP o correu
  • Còpia de seguretat i restauració de les dades
  • Protecció per contrasenya
  • Formats d'exportació: QIF i CSV (MS Excel)
  • Calculadora
  • Transaccions dividides
