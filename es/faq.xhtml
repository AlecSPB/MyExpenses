Preguntas más frecuentes

1. ¿Cómo puedo insertar valores monitarios? No encuentro la coma (,) en el teclado.
2. ¿Puedo transferir el archivo QIF a Google Docs o Dropbox?
3. Cuando creo una nueva transacción, falta la casilla para establecer el método de pago. ¿Por qué?
4. ¿Cómo puedo restaurar una copia de seguridad?

1.

¿Cómo puedo insertar valores monitarios? No encuentro la coma (,) en el teclado.

En las distintas lenguas se usan delimitadores decimales distintos: En la mayoría de los países europeos es la coma, en cambio se usa un punto en la mayoría de los países anglófonos. Desafortunadamente existe todavía un problema sin resolver, que complica el uso de la coma para la inserción de cifras decimales, y en algunos casos incluso lo imposibilita. Si usted da con este problema, puede determinar el punto como delimitador decimal en la configuración.

2.

¿Puedo transferir el archivo QIF a Google Docs o Dropbox?

Sí, active Enviar la exportación QIF en las configuraciones y deje en blanco Cargar URL de exportación QIF. Así puede elegir después de cada exportación la aplicación (por ejemplo Google Docs o Dropbox) mediante la cual desea transferir el archivo.

3.

Cuando creo una nueva transacción, falta la casilla para establecer el método de pago. ¿Por qué?

Métodos de pago están asignados a los tipos de cuenta. Inicialmente métodos de pago sólo están definidos para cuentas bancarias, no para cuentas de efectivo. Cambie el tipo de cuenta en Configuraciones → Cuenta actual, o defina un nuevo método de pago, como se explica en el manual (Métodos de pago).

4.

¿Cómo puedo restaurar una copia de seguridad?

Cuando usted crea una copia de seguridad, es guardada en la carpeta myexpenses de la tarjeta SD. Se restablece automáticamente cuando usted desinstala y reinstala Mis gastos.
