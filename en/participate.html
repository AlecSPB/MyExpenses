Contribute

By installing My Expenses Contrib, you help keeping My Expenses ad free and open source. Starting with version 1.7.0, the following new features can be tried five times, and will be unlocked for unrestricted use once you install the Contrib app:

  • Restore backups without uninstall/reinstall
  • Aggregate financial situation in account list
  • Editing of templates.
  • Reset all accounts in batch.
  • Security question for password loss
  • Cloning of transactions

If you participate by any other means, as explained below, you can request a contrib key from the preferences screen, which will also unlock all premium features.

Participate

Working on My Expenses is fun and rewarding through your appreciative and constructive feedback. Leave a comment in Google Play, or add your comment to one of the posts in the news section.

My Expenses evolves through your reports and suggestions. If you find a problem or think of an improvement, create a new issue, in Github's Issue Tracker.

You can also help translate My Expenses into new languages (e.g. {{site.missing_langs}}, ...): The user interface can be translated through Transifex, the manual is maintained in one Dockbook file.

Source code

My Expenses is free software. You can install and use it without paying, but you can also download its source code from mtotschnig/MyExpenses, inspect how it works, maybe learn from it, improve it, even redistribute it, provided you respect the licence.
