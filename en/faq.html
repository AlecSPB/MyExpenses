Frequently asked questions

1. How do I enter amounts? There is no comma (,) on the numeric keypad.
2. Can I send the QIF file to Google Docs or Dropbox?
3. While creating a new transaction, there is no field for setting the payment method. Why?
4. How can I restore a backup?

1.

How do I enter amounts? There is no comma (,) on the numeric keypad.

There are different decimal separators used in different languages: comma in most European countries, point in most English speaking countries. Unfortunately there is a still unresolved problem in the Android OS, which does not handle the comma well in its numeric input method, and in some cases even does not allow at all to enter a comma. If you encounter this problem, you can set the point as decimal separator in the Settings.

2.

Can I send the QIF file to Google Docs or Dropbox?

Yes, in the settings, activate Share QIF Export and leave Upload URL QIF Export empty. After each export, you will see a chooser for selecting an app (like Google Docs or Dropbox) capable of sending the file.

3.

While creating a new transaction, there is no field for setting the payment method. Why?

Payment methods are defined for account types. By default, there are no payment methods for cash accounts, only for bank accounts. You can change an account's type under Settings â†’ Current account, or define a new payment method as explained in the manual (Payment methods).

4.

How can I restore a backup?

When you create a backup, it is stored in the myexpenses folder of your SD card. It will be detected when you uninstall and reinstall My Expenses, and you can choose to restore it. Since version 1.7.0, you can also trigger the restore from the Settings screen (Contrib).
