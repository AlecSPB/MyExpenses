Modèles

Les modèles sont toujours le fruit d'une opération existante à travers le Créer le modèle entrée dans le menu contextuel. Vous devez entrer un titre qui permettra d'identifier le modèle.

Les modèles sont enumérés dans le menu Nouveau. Par un clic court, vous créez une nouvelle opération pour la sauver immédiatement, un click long ouvre le forumulaire pour éditer ou annuler l'opération.
