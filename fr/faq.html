Foire aux questions

1. Comment puis-je saisir les montants ? Il n'y a pas de virgule (,) sur le pavé numérique.
2. Puis-je envoyer le fichier QIF vers Google Docs ou Dropbox?
3. Lors de la création d'une nouvelle transaction, il n'existe pas de champ pour le mode de paiement. Pourquoi?
4. Comment puis-je récupérer ma sauvegarde ?
5. Clé Contrib, qu'est-ce que c'est?

1.

Comment puis-je saisir les montants ? Il n'y a pas de virgule (,) sur le pavé numérique.

Il existe différents séparateurs décimaux utilisés dans différentes langues: virgule dans la plupart des pays européens, le point dans les pays anglophones. Malheureusement, il existe un problème non encore résolu dans le système d'exploitation Android, il ne gère pas bien la virgule dans sa méthode d'entrée numérique, et dans certains cas ne permet même pas du tout d'entrer une virgule. Si vous rencontrez ce problème, vous pouvez définir le point comme séparateur décimal dans les paramètres.

2.

Puis-je envoyer le fichier QIF vers Google Docs ou Dropbox?

Oui, dans les paramètres, activez Envoyer le fichier d'export QIF, et gardez URL où sera téléchargé le fichier QIF vide. Ainsi, après chaque export, vous pourrez choisir dans un menu une apps (par exemple Google Docs ou Dropbox) capable de transférer le fichier.

3.

Lors de la création d'une nouvelle transaction, il n'existe pas de champ pour le mode de paiement. Pourquoi?

Les modes de paiement sont définis par types de comptes. Par défaut, il n'existe pas de méthodes de paiement pour les comptes de caisse, uniquement pour les comptes bancaires. Vous pouvez modifier le type du compte par Paramètres → Compte en cours, ou définir un nouveau mode de paiement comme expliqué dans le manuel (Méthodes de paiement).

4.

Comment puis-je récupérer ma sauvegarde ?

Lorsque vous créez une sauvegarde, celle-ci est stockée dans le répertoire myexpenses de votre carte SD. Elle est détectée lorsque vous désinstallez puis réinstallez Mes dépenses, et vous pouvez choissir de la recupérer. À partir de la version 1.7.0., la recupération de la sauvegarde peut aussi être initiée à partir des paramètres (Premium).

5.

Clé Contrib, qu'est-ce que c'est?

Mes dépenses est distribué avec quelques fonctionnalités Premium qui peuvent être essayées cinq fois, puis peuvent être déverrouillées par l'installation d'une app contrib payante de Google Play. Éventuellement il faut lancer Mes dépenses une fois à partir de l'écran d'accueil de la contrib app, afin que l'installation de celle-ci soit pris en compte. Mais vous n'avez pas besoin de clé contrib dans ce cas. Une clé contrib est un mécanisme permettant de déverrouiller les fonctionnalités Premium sans installation de l'app contrib. J'envoie cette clé aux utilisateurs qui contribuent au développement de Mes dépenses par d'autres moyens, par exemple la traduction de l'interface utilisateur ou en testant les pré-versions. La clé est calculée sur la base de l'Android-ID de votre appareil, que vous pouvez me communiquer par l'entrée Demandez une clé contrib dans l'écran Paramètres.
