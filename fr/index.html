Sommaire

  • Gardez une trace de vos dépenses et revenus.
  • Gérez-les directement sur votre smartphone.
  • Exportez-les vers un logiciel de finance, tel que Grisbi, Gnucash, Quicken ou MS Money.
  • Fonctionne sous toutes les versions d'Android de 2.1 à 4.3.

Fonctionnalités

Gratuit

  • Comptes multiples avec virement.
  • Créer des opérations à partir de modèles
  • Deux niveaux de catégories (importable à partir du fichier Grisbi).
  • Téléchargement du fichier export (par email, FTP, Dropbox, ...)
  • Sauvegarde et rétablissement des données
  • Protection par mot de passe
  • Formats d'export: QIF et CSV (MS Excel)
  • Calculatrice
  • Regroupement par jour, semaine, mois, année, et affichage des totaux par groupe {% include new.html %}

Premium

Les fonctionnalités suivantes peuvent être essayées cinq fois et seront deverrouillées pour une utilisation illimitée si vous installez Mes dépenses Contrib :

  • Restauration de la sauvegarde sans déinstallation/reinstallation
  • Éditer les modèles
  • Dans la liste des comptes, affichache des sommes totales
  • Réinitialiser tous les comptes en block
  • Question de sécurité en cas d'oubli du mot de passe
  • Cloner les transactions
  • Opérations ventilées {% include new.html %}
