Contribuer

En installant Mes dépenses Contrib, vous aidez à ce que Mes Dépenses soit offert sans publicité et en open source. À partir de la version 1.7.0, les nouvelles fonctionnalités suivantes peuvent être essayées cinq fois. et seront deverrouillées pour une utilisation illimitée si vous installez l'app Contrib:

  • Restauration de la sauvegarde sans déinstallation/reinstallation
  • Éditer les modèles
  • Dans la liste des comptes, affichache des sommes totales
  • Réinitialiser tous les comptes en block.
  • Question de sécurité en cas d'oubli du mot de passe
  • Cloner les transactions

Si vous participez par d'autres moyens, comme expliqué ci-dessus, vous pouvez demander une clé Contrib à partir de l'écran de paramètres. Celle-ci permettra également l'accès à toutes les fonctionnalités Premium.

Participer

Développer My Expenses, c'est plaisant et vaut la peine à travers votre feedback reconnaissant et constructif. Exprimez votre avis dans Google Play, ou ajoutez votre commentaire aux billets de la rubrique news.

My Expenses évolue à travers votre retour d'expérience et vos suggestions. Si vous rencontrez un problème ou imaginez une amélioration, ajoutez une entrée dans le Issue Tracker de Github.

Si vous êtes intéressé à aider à tester les versions à venir, rejoignez le Google groupe My Expenses Testing.

Vous pouvez aider à traduire My Expenses dans d'autres langues (par exemple: {{site.missing_langs}}, ...): Pour la traduction de l'interface usager, vous pouvez utiliser Transifex, le manual est saisi dans un fichier Dockbook.

Code source

My Expenses is free software. You can install and use it without paying, but you can also download its source code from mtotschnig/MyExpenses, inspect how it works, maybe learn from it, improve it, even redistribute it, provided you respect the licence.
