Summary

My Expenses is an Android application designed to keep track of your expenses and incomes, and to export them as QIF files into a desktop finance tool, like Grisbi (Open Source), Gnucash (Open Source), MS Money, Quicken or Wiso Mein Geld.
Requires Android 2.0 and above.

Features:

  • multiple accounts with transfers.
  • two levels of categories (import from Grisbi XML)
  • upload QIF file via FTP or email
  • data backup and restore

Sourcecode

My Expenses is free software. Free like in "free beer", i.e. you can install it without paying. But also free like in "free speech", ie. you can download its source code from mtotschnig/MyExpenses, inspect how it works, maybe learn from it, improve it, even redistribute it.
