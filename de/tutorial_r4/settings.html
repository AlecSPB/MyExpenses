Einstellungen

Der Bildschirm Einstellungen besteht aus fünf Teilen :

Verwalten

Hier haben Sie Zugriff auf alle von Meine Ausgaben verwalteten Objekte: Konten, Vorlagen, Kategorien, Parteien und Zahlungsweisen.

Benutzerschnittstelle

Einstellungen für das Verhalten der Benutzerschnittstelle.

Import

Hier können Kategorien und Parteien aus Grisbi importiert werden.

Export

Für den QIF Export kann hier der Zeichensatz konfiguriert werden, und eingestellt werden, ob und wie dieses übertragen werden soll.

Sicherungskopie

Erstellung einer Sicherungskopie der Datenbank und der Einstellungen auf der SD-Karte.

Verwalten

Parteien

Auf diesem Bildschirm wird die Liste der Parteien angezeigt, die für die automatische Komplettierung des Feldes Zahler/Empfänger verwendet wird. Sie können Einträge aus dieser Liste entfernen, indem Sie das Markierungsfeld in der jeweiligen Zeile ankreuzen, und dann den Schaltknopf Löschen betätigen.

Zahlungsweisen

Auf diesem Bildschirm wird die Liste der vordefinierten Zahlungsweisen angezeigt (Scheck, Kreditkarte, Einzahlung, Abbuchung) und Sie können hier auch neue Zahlungsweisen definieren. Wenn Sie auf einen Eintrag in der Liste klicken (oder Neu auswählen), gelangen Sie zu einem Formular, in dem Sie eine Bezeichnung (außer für die vordefinierten) und den Typ (handelt es sich um eine Zahlungsweise für Einnahmen, Ausgaben oder neutral) festlegen, und die Kontotypen auswählen, für die die Zahlungsweise anwendbar ist.

Benutzerschnittstelle

Stil

Sie können zwischen einem hellen und einem dunklen Stil wählen.

Androids Standardmenü verwenden

Indem Sie diese Option anklicken, können Sie anstatt der Symbolleiste Androids Standardmenü verwenden, um Befehle auszuführen. Sie können diesen Parameter auch durch Drücken des Hardware Menüknopfes verändern.

Sortierreihenfolge der Kategorien

Standardmäßig werden die Kategorien anhand der Benützungshäufigkeit sortiert (wie oft werden sie einer Buchung zugeordnet). Entfernen Sie das Häckchen von dieser Option, wenn Sie möchten, dass die Kategorien immer in alphabetischer Reihenfolge aufgelistet werden.

Dezimaltrennzeichen für die Eingabe von Währungsbeträgen

Standardmäßig wird für Währungsbeträge in Buchungen (oder das Anfangssaldo eines Kontos) das Dezimaltrennzeichen Ihrer Spracheinstellung verwendet (Komma in den meisten europäischen Ländern, Punkt in den meisten englischsprachigen Ländern)[3], hier kann dieses explizit konfiguriert werden. Sie können hier aber auch festlegen, dass Sie Untereinheiten (z.B $ Cents, € Cent, oder £ Pence) als ganze Zahlen eingeben.

Import

Der Eintrag Import aus Grisbi erlaubt die in Grisbi definierten Kategorien (Englisch, Deutsch oder Französisch) zu importieren. Sie können auch eine Grisbi Datei (in XML) verwenden, die Ihre eigenen Kategorien enthält. Sie können diese Datei editieren und alle Informationen außer die Elemente Category und Sub_category entfernen[4]. Kopieren Sie diese Datei unter dem Namen grisbi.xml in einen Ordner myexpenses in das Wurzelverzeichnis der SD-Karte[5]. Sie können zusätzlich auch die von Grisbi gespeicherten Parteien (Empfänger und Zahler), aus dieser Datei importieren (Element Party). Wenn Sie Ihre eigene Datei als Import-Quelle auswählen, können Sie entweder Parteien und Kategorien oder nur Kategorien importieren. Die Partner werden im Buchungsformular verwendet, um Ihnen für das Feld Empfänger/Zahler ein automatische Ausfüllen vorzuschlagen.

Export

Kreuzen Sie die Option QIF Export versenden an, wenn Sie die QIF Datei nach dem Export Ihrer Buchungen an eine entfernte Adresse übertragen wollen. Unter Upload URL QIF Export können Sie das Ziel konfigurieren:

  • Die URL eines Ordners, auf den Sie über FTP Schreibzugriff haben (z.B. "ftp://login:password@my.example.org:port/my/directory/"). Falls eine solche auf Ihrem Gerät installiert ist, wird Meine Ausgaben eine externe App, z.B. Mit FTP versenden aufrufen.

  • Eine Email URL (z.B. "mailto:john@my.example.com"). Ihre Email-Anwendung wird gestartet, und eine Email erzeugt mit der Datei als Anhang.

  • Sie können dieses Feld auch leer lassen, dann können Sie jedesmal zwischen auf Ihrem Gerät installierten Apps wählen, die Dateien übertragen können (z.B Dropbox, Google Docs, …).

Backup und Wiederherstellung

Sicherungskopie erzeugt eine Sicherungskopie von Datenbank und Einstellungen im myexpenses Ordner auf der SD-Karte. Die Datenbank wird in der Datei BACKUP gespeichert, die Einstellungen in der Datei BACKUP_PREF. Beide werden automatisch wiederhergestellt, wenn Meine Ausgaben neu installiert wird. Sie können also die Anwendungsdaten komplett auf ein weiteres Gerät kopieren, indem Sie diese beiden Dateien an denselben Ort auf diesem kopieren, bevor Sie Meine Ausgaben installieren.


[3] Meine Ausgaben verwendet auf Geräten mit Android 2.3. als Standard den Punkt, da diese am meisten von dem in der FAQ beschriebenen Problem betroffen sind.

[4] Meine Ausgaben unterstützt auch das Grisbi Datei Format 0.5.0, in dem die Elemente Categorie und Sous_categorie heißen (Tiers für Parteien).

[5] Natürlich können Sie auch Ihre eigene XML Datei erzeugen. Sie finden Beispiele dafür auf Github.
