Häufig gestellte Fragen

1. Wie kann ich Beträge eingeben? Auf der numerischen Tastatur finde ich kein Komma (,).
2. Kann ich die QIF Datei auf Google Docs oder Dropbox übertragen?

1.

Wie kann ich Beträge eingeben? Auf der numerischen Tastatur finde ich kein Komma (,).

In den verschiedenen Sprachen werden unterschiedliche Dezimaltrennzeichen verwendet: Komma in den meisten europäischen Ländern, Punkt in den meisten englischsprachigen Ländern. Leider gibt es unter Android ein noch ungelöstes Problem, das die Verwendung des Kommas zur Eingabe von Dezimalzahlen erschwert, in manchen Fällen sogar unmöglich macht. Wenn Sie diesem Problem begegnen, können Sie in den Einstellungen den Punkt als Dezimaltrennzeichen festlegen.

2.

Kann ich die QIF Datei auf Google Docs oder Dropbox übertragen?

Ja, aktivieren Sie QIF Export versenden in den Einstellungen, und lassen Sie Upload URL QIF Export leer. Dann können Sie nach jedem Export die App (wie z.B. Google Docs oder Dropbox) auswählen, mittels deren Sie die Datei übertragen wollen.
