Konten

Beim ersten Aufruf von Meine Ausgaben wird ein Standardkonto angelegt, und Sie können sofort mit der Eingabe von Buchungen beginnen. Falls Sie das Standardkonto verändern wollen, verwenden Sie Einstellungen → Aktuelles Konto. Mittels Konten → Neu … hingegen können Sie ein neues Konto anlegen.

Geben Sie eine Bezeichnung, eine Beschreibung, den Anfangssaldo[1] und die Währung des Kontos ein. Die Währung muss anhand ihrer dreistelligen Abkürzung ISO 4217 eingegeben werden, aber kann durch Klick auf Auswählen einfach aus einer Liste ausgewählt werden.

Sie können einem Konto einem der folgenden Typen zuordnen: Bargeldkonto, Bankkonto, Anlagekonto, Kreditkonto. Für Kontotypen sind Zahlungsweisen definiert, wie in Zahlungsweisen beschrieben. Hier wählen wir den Typ als Bankkonto, um zu zeigen, wie Zahlungsweisen verwendet werden.

Nach Bestätigung der Eingabe gelangen Sie wieder zur Liste der Buchungen. Wenn Sie ein neues Konto angelegt haben, wird dieses automatisch zum aktiven Konto. Der Schaltknopf Konten erlaubt das Wechseln zwischen Konton. Ein langer Klick öffnet ein Menü mit allen Konten zur Auswahl. Ein einfacher Klick ruft entweder nacheinander alle Konten auf, oder wechselt zum zuletzt aktiven Konto, oder , abhängig von der Einstellung Verhalten der Schaltfläche "Konten".

Konten → Zusammenfassung öffnet eine Übersicht aller Konten mit dem aktuellen Finanzstatus. Durch den Klick auf ein Konto wird das Konfigurationsformular geöffnet. Ein langer Klick bringt das Kontextmenü zum Vorschein, mit der Möglichkeit, das Konto zu löschen, mit der Ausnahme des aktuellen Kontos, das nicht gelöscht werden kann.
