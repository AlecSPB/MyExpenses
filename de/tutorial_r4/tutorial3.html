Kategorien

Zu Beginn gibt es noch keine Kategorien. Es gibt zwei Möglichkeiten, diese anzulegen.

Der Schaltknopf Kategorien importieren erlaubt Kategorien aus verschiedenen Quellen zu importieren wie in Import aus Grisbi erklärt. Dieser Schaltknopf ist nur sichtbar solange es keine Kategorien gibt, aber Sie können jederzeit im Bildschirm Einstellungen auf die Import-Funktionalität zugreifen.

Sie können die importierten Kategorien und Unterkategorien jetzt Ihren Buchungen zuordnen. Als primäres Sortierkriterium wird standardmäßig die Anzahl der Verwendungen einer Kategorie verwendet (sekundäres Kriterium ist die alphabetische Reihenfolge). Dadurch haben Sie zu den am häufigsten verwendeten Kategorien am schnellsten Zugriff. Wenn Sie die Kategorien lieber alphabetisch geordnet wollen, können Sie die Einstellung Sortierreihenfolge der Kategorien deaktivieren.

Die zweite Möglichkeit besteht darin, Kategorien selbst anzulegen. Neue Hauptkategorie öffnet einen Dialog zur Eingabe einer Bezeichnung.

Ein Klick auf eine Hauptkategorie öffnet die Liste der Unterkategorien. Die Zuordnung einer Unterkategorie zur aktuellen Buchung erfolgt durch kurzen Klick. Die Zuordnung einer Hauptkategorie ist durch das Kontextmenü möglich.

Dieses erlaubt auch das Anlegen neuer Unterkategorien, das Löschen und das Verändern einer Hauptkategorie. Die Unterkategorien verfügen auch über ein Kontextmenü mit Einträgen zum Löschen und Verändern.
