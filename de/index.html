Zusammenfassung

  • Halten Sie über Ihre Einnahmen und Ausgaben Buch.
  • Verwalten Sie diese direkt auf Ihrem Smartphone.
  • Exportieren Sie diese in ein Desktop-Finanzprogrammm wie Grisbi (Open Source), Gnucash (Open Source), Quicken oder MS Money).
  • Läuft auf allen Android Versionen von 2.1. bis 4.3

Funktionalitäten

Gratis

  • Mehrere Konten mit Umbuchungen.
  • Buchungen aus Vorlagen erzeugen
  • Zwei Ebenen von Kategorien mit Import aus Grisbi XML Datei.
  • Übertragung der Export Datei (über E-Mail, FTP, Dropbox, ...)
  • Daten Backup und Wiederherstellung
  • Passwortschutz
  • Export-Formate: QIF und CSV (MS Excel)
  • Taschenrechner
  • Buchungen pro Tag, Woche, Monate, Jahr gruppieren und Summen pro Gruppe anzeigen {% include new.html %}

Premium

Folgende Funktionen können fünfmal ausprobiert werden, und werden durch die Installation von Meine Ausgaben Contrib freigeschaltet :

  • Darstellung der Gesamtsummen in der Kontenübersicht
  • Wiederherstellung der Sicherungskopie ohne Deinstallierung/Reinstallierung
  • Vorlagen verändern
  • Alle Konten auf einmal zurücksetzen
  • Sicherheitsfrage für Passwortverlust
  • Klonen von Buchungen
  • Splitbuchungen {% include new.html %}
