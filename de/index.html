Zusammenfassung

  • Halten Sie über Ihre Einnahmen und Ausgaben Buch.
  • Verwalten Sie diese direkt auf Ihrem Smartphone.
  • Exportieren Sie diese in ein Desktop-Finanzprogrammm wie Grisbi (Open Source), Gnucash (Open Source), Quicken oder MS Money).
  • Läuft auf allen Android Versionen von 2.0. bis 4.2

Funktionalitäten

  • Mehrere Konten mit Umbuchungen.
  • Buchungen aus Vorlagen erzeugen
  • Zwei Ebenen von Kategorien mit Import aus Grisbi XML Datei.
  • Übertragung der Export Datei (über E-Mail, FTP, Dropbox, ...)
  • Daten Backup und Wiederherstellung
  • Passwortschutz
  • Export-Formate: QIF und CSV (MS Excel)
